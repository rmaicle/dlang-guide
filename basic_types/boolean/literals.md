There are only two possible boolean literals; the [reserved identifiers] `true` and `false`.

{% highlight d linenos %}
bool a = true;
bool b = false;
bool c = 1;
bool d = 0;
writeln(a);
writeln(b);
writeln(c);
writeln(d);
{% endhighlight %}

See [Integral Literals].

[reserved identifiers]: /dlang-guide/identifiers/reserved/index.html
