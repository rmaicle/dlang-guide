Allowed operations on boolean types are:

| Operator     | Description |
|:------------:|-------------|
| &            | bitwise AND |
| &#124;       | bitwise OR  |
| ^            | bitwise XOR |
| &=           | bitwise AND then assign |
| &#124;=      | bitwise OR then assign  |
| ^=           | bitwise XOR then assign |
| !            | Negation    |
| &&           | Logical AND |
| &#124;&#124; | Logical OR  |
| ? :          | Conditional operator |
