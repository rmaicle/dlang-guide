###### Header File Options

<div markdown='1' class='syntax'>
    dmd files [...] -H
    dmd files [...] -Hd<path>
    dmd files [...] -Hf<filename>

`-H`
: Generate _header_ file.

`-Hd<path>`
: Write _header_ file to `path`.

`-Hf<filename>`
: Write _header_ to `filename`.
</div>
