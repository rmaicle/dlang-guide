###### Color Options

<div markdown='1' class='syntax'>
    dmd files [...] -color
    dmd files [...] -color=on
    dmd files [...] -color=off

Show colored console output. The default depends on terminal capabilities. 

`-color`
`-color=on`
: Force colored console output.

`-color=off`
: No colored console output.
</div>
