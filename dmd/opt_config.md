###### Configuration

Use configuration file at specified path.

<div markdown='1' class='syntax'>
    dmd -conf=<path>

`path`
: A relative or absolute path to the configuration file.
  The configurationn file is dmd.conf.
  In Linux, by default, dmd.conf is in /etc.
</div>
