###### Symbolic Debug Information Option

<div markdown='1' class='syntax'>
    dmd files [...] -g
    dmd files [...] -gc

`-g`
: Add symbolic debug information for D debuggers.

`-gc`
: Add symbolic debug information for non-D debuggers.
</div>
