###### Import Options

<div markdown='1' class='syntax'>
    dmd files [...] -I<path>
    dmd files [...] -J<path> [...]

`-I<path>`
: Specify path where compiler will look for imports.

`-J<path>`
: Specify path where compiler will look for string imports.
</div>
