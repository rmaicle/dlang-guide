###### Code Coverage Options

<div markdown='1' class='syntax'>
    dmd files [...] -cov
    dmd files [...] -cov=nnn

`-cov`
: Perform code coverage analysis.

`-cov=nnn`
: Perform code coverage analysis and require at least `nnn`% code coverage.
</div>
